# Number Guessing Game

##  Project Description

A Python command-line game where the player guesses a randomly generated number. The game offers multiple difficulty levels, hints, scoring, and a game summary.

##  Features

- Easy, Medium and Hard modes
- Random Number Generation
- Input Validation
- Score System
- Attempts Counter
- "Too High" / "Too Low" Hints
- Game Summary
- Play Again Option

##  Technologies Used

- Python 3
- Random Module
- VS Code

##  Project Structure

Number-Guessing-Game/
│── main.py
│── README.md

##  How to Run

Run:

```bash
python main.py
```
